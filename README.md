# Diabetes Predictor

## Overview

This project is a machine learning-based web application that predicts whether a person is likely to have diabetes based on certain medical input parameters. It is designed to provide a simple interface where users can enter values and get a prediction instantly.

## Features

* Takes user input such as glucose level, BMI, age, etc.
* Predicts the likelihood of diabetes using a trained model
* Simple and easy-to-use interface
* Quick prediction results

## Tech Stack

* Frontend: HTML, CSS
* Backend: Python
* Libraries: NumPy, Pandas, Scikit-learn

## How It Works

The model is trained on a dataset containing medical attributes related to diabetes. Based on the input values provided by the user, the model processes the data and returns a prediction indicating whether the person is likely to be diabetic or not.
